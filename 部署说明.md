# 🖼 图片拼接字段捷径 - 完整部署指南

飞书多维表格 FaaS 版字段捷径，**一次配置，每行自动批量合成拼图**。

---

## 📦 项目结构

```
├── faas/                   # 字段捷径（飞书 FaaS 沙箱运行）
│   ├── src/index.ts        # 捷径入口
│   ├── package.json
│   └── tsconfig.json
│
└── collage-server/         # 合图服务（Railway 部署）
    ├── server.js
    ├── package.json
    └── railway.toml
```

---

## 🚀 第一步：部署合图服务到 Railway

> FaaS 沙箱不支持 canvas/sharp，需要外部服务合图，FaaS 只负责调度。

### 操作步骤

1. 将 `collage-server/` 下三个文件上传到 GitHub 新仓库
2. 打开 [railway.app](https://railway.app)，**New Project → Deploy from GitHub**
3. 选择该仓库，Railway 自动识别为 Node.js 项目并部署
4. 部署完成后：**Settings → Networking → Generate Domain**
5. 访问 `https://你的地址/health`，返回 `{"status":"ok"}` 即成功

---

## 🔧 第二步：发布字段捷径

```bash
cd faas
npm install       # 安装依赖（需要 Node.js 14.16+）
npm run start     # 本地调试
npm run pack      # 打包，生成 output/output.zip
```

将 `output.zip` 上传到飞书字段捷径平台发布。

---

## 📖 第三步：在多维表格中使用

1. **+ 新建字段 → 字段捷径**，搜索插件名
2. 填写配置：
   - 图片来源字段（附件）
   - 拼接布局：智能网格 / 2列 / 3列 / 横排
   - 输出尺寸：1080×1080 / 1200×900 / 1920×1080 / 1080×1920
   - 图片间距（默认 8px）
   - 背景颜色（默认 #ffffff）
   - **合图服务地址**：填写 Railway 部署地址
3. 保存后，所有行自动批量生成拼图 ✅

---

## 📐 布局说明

| 值 | 说明 |
|----|------|
| `smart` | 智能：1张=1列，2张=2列，3-4张=2×2，5-9张=3列，10+张=4列 |
| `2col` | 固定2列，保持原始比例居中 |
| `3col` | 固定3列，保持原始比例居中 |
| `row` | 横排一行 |

## ❓ FAQ

**Q: 为什么用 Telegraph 图床？**
A: 免费、无需注册、飞书可直接下载。如需换成阿里云/腾讯云 OSS，修改 `server.js` 中的 `uploadToTelegraph` 函数即可。

**Q: Railway 免费够用吗？**
A: 每月 $5 免费额度，合图服务 CPU 消耗低，正常使用完全够用。
